O Software abrira um menu:
Onde pressionando: 
"0" =  Exit 
"1" = Mostrara o vértice, e seu respectivo grau;
"2" = Todos os cliques maximais;
"3" O Coeficiente de Aglomeração de cada vértice e o Coeficiente médio de Aglomeração do Grafo.
